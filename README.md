# Loopback Vue Starter Kit
Loopback, Vue 2.0, vue-router and vuex starter kit for server side rendering with webpack 2.0

The project is generated by [LoopBack](http://loopback.io).

## Build Setup

``` bash
npm install
npm start
```

## Development Setup

```bash
npm run watch-c
npm run watch-s
node server/server.js
```
